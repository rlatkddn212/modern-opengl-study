#### 그래픽스 파이프라인

- 파이프라인 분할을 사용하여 성능을 향상시켰다.
- 추상화 레이어를 통해 어떤 그래픽 카드인지 상관없이 사용가능하다.
- 쉐이더를 이용하여 프로그래밍 가능한 스테이지와 고정 함수(프로그래밍 불가능한) 스테이지로 구분된다.
